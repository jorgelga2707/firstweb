Let's learn to insert a tweet in a web page:

  1. First we go to our navigator.
  2. Next we go to https://twitter.com .
  3. Next make click in the ... icon on the tweet.
  4. Make click in the embedded tweet option.
  5. Costumize the tweet how you like Dark Theme or light Theme.
  6. Now make click on copy code icon.
  7. Copy the code in your tweets web page.
